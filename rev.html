​Mequanent Degu Belete is an Ethiopian computer engineer, lecturer, and researcher affiliated with Debre Markos University. His work primarily focuses on natural language processing (NLP), machine learning, and computer vision, with a particular emphasis on Ethiopian languages such as Amharic and Ge'ez.

Education:

Bsc Degre in Electrical and Computer Engineering (Electronic Communications), graduated in 2015 from Debre Markos University

Msc Degre in Computer Engineering graduated in 2022 from Debre Markos University. Further information Mequanent Degu

Research Interest:

His research encompasses:

    Natural language processing
    Machine Learning
    Computer Vision
    Deep Learning
    Feature Extraction
    Data Mining​

He has a particular interest in developing computational tools for Ethiopian languages, aiming to enhance language processing capabilities for languages like Amharic and Ge'ez.

Selected Publications

Amharic Language Hate Speech Detection System from Facebook Memes Using Deep Learning: This study developed a system to detect hate speech in Amharic language memes on Facebook using deep learning techniques. SSRN

Enhancing Word Sense Disambiguation for Amharic Homophone Words Using Bidirectional Long Short-Term Memory Network: This research focused on improving the disambiguation of Amharic homophones using BiLSTM networks. ResearchGate

He has also developed an Amharic language text stemmer in Python, which is available on his GitHub repository.
